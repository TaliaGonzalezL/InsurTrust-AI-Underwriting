# 🛡️ InsurTrust AI: Intelligent Medical Underwriting
**Transforming Insurance Risk into Predictive Certainty**

InsurTrust AI is an Agentic platform designed to automate and fortify the medical underwriting process. By combining **Random Forest Classification** with **Agentic Reasoning (LangChain + Gemini)**, the system identifies high-risk profiles and atypical claims patterns that traditional actuarial tables often overlook.

### 🚀 Key Features
*   **0.90 F1-Score Predictor:** A robust Machine Learning pipeline optimized for high-stakes risk detection.
*   **Agentic Analysis:** An autonomous AI agent that interrogates complex medical datasets via natural language.
*   **Resilience Architecture:** Custom sanitization layers to ensure production-grade stability and data integrity.
